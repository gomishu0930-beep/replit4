## 2. 絶対禁止ルール

### 二刀流NG（永久禁止）
- ❌ 相互RT・相互紹介
- ❌ 同時刻投稿（6時間以上の時差必須）
- ❌ 同一IPでの同時運用
- ❌ 同じアフィリIDの同日使用

### 凍結直結NG
- ❌ 投稿本文にアフィリリンクを直接貼る
- ❌ リプライ内に自分の投稿URLを含める
- ❌ 1時間に3件以上のフォロー
- ❌ #相互フォロー 系タグの使用
- ❌ 過激ジャンル（レイプ・盗撮・無修正等）
- ❌ サンプル動画をそのまま投稿
- ❌ フォロバ待ち48h未満でのアンフォロー

## 4. 投稿ルール

### コンテンツ配分

| 種別 | 割合 | 内容 | リンク |
|---|---|---|---|
| **Poll（投票）** | 70% | VS形式の2〜4択 | なし |
| **セールレビュー** | 20% | 投票結果と連動した作品紹介 | 自己リプでURL後置き |
| **MyFans紹介** | 10% | 投票結果に紐づけた自然な紹介 | プロフリンクへ誘導 |

### Poll投稿テンプレート

```

```
→ 選択肢: 最大4択

| 曜日 | テーマ | 例 |
|---|---|---|
| 月 | ジャンル対決 | 巨乳 vs 美乳 vs 貧乳 |
| 火 | シチュ対決 | 不倫 vs 寝取られ |
| 水 | 属性対決 | JD vs OL vs 人妻 |

| 土 | ランキング | 歴代最高AV女優は？ |

1. 投稿本文にリンクは**絶対に入れない**

4. EV5<3の投稿にはリンクを付けない

## 5. フォロー戦略

### 1日のフォロー上限（計算式）
```
上限 = min(6, ⌊フォロワー数 × 0.30⌋)

例:
  フォロワー  0人 → 上限 0（しない）
  フォロワー  5人 → 上限 1
  フォロワー 10人 → 上限 3
  フォロワー 20人 → 上限 6（最大）
```
※ 初日〜3日目のみ特例で「1日3フォロー」を許可

### フォロー対象の条件（全て満たすこと）
- フォロワー 50〜300人
- 直近7日以内に投稿あり
- プロフにAV/エロ/R18/FANZA/アダルトのいずれかを含む
- センシティブ設定ON
- botではない（手動投稿が確認できる）

### フォロー手順
```
Step 1: 対象の直近ツイートにいいね ❤
Step 2: 30分後にフォロー
```
※ リプ→フォロー、フォロー→即DM は禁止

### アンフォロールール
| ルール | 内容 |
|---|---|
| 待ち期間 | 48時間 |
| 判定 | 48h後フォロバなし → アンフォロー |
| 1日の上限 | 12件 |
| 間隔 | 2時間以上 |
| 再フォロー | 一度外した相手は永久禁止 |

## 用語集

| 用語 | 意味 |
|---|---|
| **SBI** | Shadowban Index。0=正常、数値が高いほど制限中 |
| **EV5** | 投稿後5分間のエンゲージメント指標 |
| **IP** | インプレッション（表示回数） |
| **FF比** | フォロー数÷フォロワー数。低いほど健全 |
| **CVR** | コンバージョン率（クリック→購入） |
| **CTR** | クリック率（表示→クリック） |
| **Poll** | X（Twitter）のアンケート機能 |
| **自己リプ方式** | 自分の投稿にリプライでリンクを貼る方法 |
