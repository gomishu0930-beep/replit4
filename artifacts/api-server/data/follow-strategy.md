## フォロー戦略

### フォロー上限の計算式（FF比動的制御）

```
1日のフォロー上限 = min(6, ⌊現フォロワー数 × 0.30⌋)

例:
- フォロワー 0人 → 上限 0（フォローしない）
- フォロワー 3人 → 上限 0（まだ早い）
- フォロワー 5人 → 上限 1
- フォロワー 10人 → 上限 3
- フォロワー 15人 → 上限 4
- フォロワー 20人以上 → 上限 6（最大）
```

**初日〜3日目の特例**: フォロワー0の状態ではフォロー上限式が0になるため、初日〜3日目のみ「1日3フォロー」の特例を適用。

### フォロー対象の選定基準

**検索条件:**
```
X検索: #FANZA lang:ja -filter:replies
→ 最新タブ → 直近24時間の投稿
```

**対象の条件（全て満たすこと）:**
- [ ] フォロワー 50〜300人
- [ ] 直近7日以内に投稿あり
- [ ] プロフィールに「AV」「エロ」「R18」「FANZA」「アダルト」のいずれかを含む
- [ ] センシティブ設定ONのアカウント
- [ ] botではない（手動投稿が確認できる）

**除外対象:**
- フォロワー300人以上（フォロバ率低い）
- フォロワー50人未満（影響力不足）
- 業者・スパムアカウント
- 直近30日間に投稿なし

### フォロー手順（いいね → フォローの2ステップ）

```
Step 1: 対象の直近ツイートにいいね ❤
Step 2: 30分後にフォロー
```

**やらないこと:**
- リプライしてからフォロー（新規アカのリプはスパム判定リスク）
- フォローと同時にDM
- 一気に複数フォロー（30分間隔を厳守）

### アンフォロールール

| ルール | 内容 |
|--------|------|
| フォロバ待ち期間 | 48時間 |
| 判定 | 48時間経過後もフォロバなし → アンフォロー |
| 1日の上限 | 最大12件 |
| 間隔 | 2時間以上空ける |
| 再フォロー | 一度アンフォローした相手は再フォロー禁止 |
| 記録 | GoogleSheetに全件記録（日時/相手ID/結果） |

## フォロワー獲得戦略

### 1. オーガニック成長（メイン施策）

| 施策 | 内容 | 期待効果 |
|------|------|---------|

| 結果発表ツイート | 「昨日は〇〇派が勝利！🏆」 | リピーター獲得 |

### 2. 相互フォロー施策

| 施策 | 内容 | 注意点 |
|------|------|--------|
| Cluster-Follow | #FANZAクラスタの中規模アカをフォロー | FF比2.5以下死守 |
| いいね先行 | フォロー前に必ずいいね1件 | フォロバ率+5〜8%向上 |
| 48h判定 | フォロバなしは48hでアンフォロー | FF比管理のため |

### 3. リプライ戦略（条件付き解禁）

**解禁条件:** フォロワー10人以上 AND SBI=0

| 項目 | 内容 |
|------|------|
| 解禁時期 | Day 8以降（条件クリア時） |
| 頻度 | 1日1本のみ |
| 対象 | フォロワー1,000人未満の#FANZA投稿 |
| 内容例 | 「この作品気になってました🙏」 |
| 禁止事項 | PollリンクをリプライにNGで入れない |
| 絵文字 | 3個以内 |
| 検証 | 初回リプから72h後に可視率チェック、20%未満なら禁止に戻す |

**Poll誘導はプロフィール固定ツイートで行う（リプ内URL絶対禁止）**

### 4. ハッシュタグ戦略

| フェーズ | ハッシュタグ | 理由 |
|---------|------------|------|
| Week 1-2 | #FANZA #エロ投票 | 最小限で安全に |
| Week 3-4 | #FANZA #エロ投票 + 投稿テーマ系1つ | 露出拡大 |
| Month 2以降 | 最大3個まで | トレンドに合わせて柔軟に |

**禁止タグ:** #相互フォロー #フォロバ100 など（スパム判定直結）

### 5. Pollからのフォロワー転換

```
投票してくれた人 → プロフ訪問 → 固定ツイートで最新Poll表示
→ 「毎日投票あるなら面白い」→ フォロー

導線: Poll投票 → 結果が気になる → プロフ訪問 → 固定ツイで今日のPoll発見 → フォロー
```

## フォロバ率のキャリブレーション

**初期想定:** フォロバ率 24%（Grok実測データ）

**Week 1終了時（Day 7 21:30 JST）に検証:**
```
実測フォロバ率 = Week 1フォロワー増加数 ÷ Week 1累計フォロー数

判定:
- 実測 ≥ 20% → Week 2計画どおり（日5-6フォロー）
- 実測 15〜19% → Week 2を日4フォローに減速
- 実測 < 15% → Week 2を日3フォローに大幅減速、戦略再検討
```
