## Phase 3: DECIDE（意思決定）

### A. アカウント名（表示名）

```
エロ投票ラボ🗳️🔞
```

**理由**:
- 「エロ」でジャンル明確化
- 「投票ラボ」で参加型＋研究感 → 知的好奇心を刺激
- @fanza_poll_lab との整合性が高い
- 🗳️で投票感を視覚化、🔞で年齢制限宣言
- 8文字以内で覚えやすい

**代替案**:
- エロ調査局🗳️🔞
- みんなのエロ投票🗳️
- エロ白書🔞📊

### F. コンテンツ戦略（70/20/10配分の具体化）

#### 70% エンゲージメント投稿 = ポール中心

**投票テンプレート（日替わりローテーション）**:

| 曜日 | 投票テーマ | 選択肢例 |
|------|-----------|---------|
| 月 | ジャンル対決 | 「巨乳vs美乳vs貧乳」 |
| 火 | シチュ対決 | 「不倫vs寝取られvsNTR」 |
| 水 | 属性対決 | 「JDvsOLvs人妻」 |
| 木 | プレイ対決 | 「正常位vs後背位vs騎乗位」 |
| 金 | 新作予想 | 「今週のFANZA1位はどれ？」 |
| 土 | ランキング | 「歴代最高AV女優は？」 |
| 日 | フリー | フォロワーリクエスト投票 |

**投票のルール**:
- 選択肢は4択以内（シンプルさ最優先）
- 投票期間: 24時間（毎日結果が出る→フォロー動機）
- テキストは30文字以内の質問＋絵文字
- 画像は使わない（投票のみでインプ最大化）

**例文**:
```
みんなはどっち派？🗳️

正直に答えてね💦
```
→ 選択肢: 巨乳 / 美乳 / 貧乳 / その他

#### 20% FANZA アフィリエイト投稿

**投票結果と連動させる**:
```
昨日の投票結果🗳️

「巨乳vs美乳」→ 巨乳派 62%！

というわけで巨乳派のみんなに捧げる
⭐4.8点の神作品がこちら👇

[画像4枚]
```

→ リプライにアフィリリンク（本文には入れない）

**ポイント**:
- 投票結果を起点にすることで「押し付け感」がゼロ
- 「みんなが選んだ」= 社会的証明が働く
- 投票→結果→レビューの自然な導線
